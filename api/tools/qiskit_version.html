qiskit_version#

Qiskit Version (qurry.tools.qiskit_version)

  • This module is for checking the version of qiskit and its packages.

  • For remaking the deprecated QiskitVersion in qiskit.version since 1.0.0.

qurry.tools.qiskit_version.KNOWN_CORE_PACKAGE = {'deprecated': ['qiskit-ibmq-provider', 'qiskit-terra', 'qiskit-ignis'], 'into-community': ['qiskit-nature', 'qiskit-finance', 'qiskit-optimization', 'qiskit-machine-learning'], 'main': ['qiskit-aer', 'qiskit-aer-gpu', 'qiskit-aer-gpu-cu11', 'qiskit-ibm-provider', 'qiskit-ibm-runtime']}#

A mapping of package names to their versions in Qiskit. This is an instance of QiskitVersion. It contains the versions of the main, deprecated, and into-community packages of Qiskit.

qurry.tools.qiskit_version.qiskit_version_v0_check()[source]#

Check the version of qiskit and its packages.

This function checks the version of qiskit and its packages, and raises a warning if the version is lower than 1.0.0. It is recommended to use the latest version of qiskit for compatibility with Qurrium.
