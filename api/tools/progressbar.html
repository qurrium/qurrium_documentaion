class qurry.tools.progressbar.tqdm(*_, **__)[source]#

A fake tqdm class for type hint.

For tqdm not yet implemented their type hint by subscript like: >>> some_tqdm: tqdm[int] = tqdm(range(10))

So, we make a fake tqdm class to make it work. And it should be tracked by this issue: tqdm/tqdm#260 To avoid the conflict, you SHOULD NOT IMPORT this class and keep it only working for qurry_progressbar() as type hint.
