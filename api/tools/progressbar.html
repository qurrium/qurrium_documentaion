progressbar#

Progress Bar for Qurrium (qurry.tools.progressbar)

For the followings GitHub issues on https://tqdm.github.io/

  • tqdm/tqdm#260 We do some improvement in our package. We make a fake tqdm class tqdm for type hint.

  • tqdm/tqdm#705 The intersphinx usage in docstring like:

    """Dummy docstring.

Hey, it's a :class:`~tqdm.tqdm`
"""

    It‚Äôs not available, so we replace them by:

    """Dummy docstring.

Hey, it's a `tqdm.tqdm <https://tqdm.github.io/>`
"""

    by the external link to the tqdm documentation, which follows Links to External Web Pages

qurry.tools.progressbar.DEFAULT_BAR_FORMAT = {'qurry-barless': '| {n_fmt}/{total_fmt} - {desc} - {elapsed} < {remaining}', 'qurry-full': '| {n_fmt}/{total_fmt} {percentage:3.0f}%|{bar}| - {desc} - {elapsed} < {remaining}', 'simple': '| {desc} - {elapsed} < {remaining}'}#

Default bar format for progress bar.

  • ‚Äúsimple‚Äù:

    A simple format with description, elapsed time, and remaining time.

    "| {desc} - {elapsed} < {remaining}"
  • ‚Äúqurry-full‚Äù:

    A full format with count, percentage, bar, description, elapsed time, and remaining time.

    "| {n_fmt}/{total_fmt} {percentage:3.0f}%|{bar}| - {desc} - {elapsed} < {remaining}"
  • ‚Äúqurry-barless‚Äù:

    A format without the bar, showing count, description, elapsed time, and remaining time.

    "| {n_fmt}/{total_fmt} - {desc} - {elapsed} < {remaining}"

class qurry.tools.progressbar.tqdm(*_, **__)[source]#

A fake tqdm class for type hint.

For tqdm not yet implemented their type hint by subscript like:

some_tqdm: tqdm[int] = tqdm(range(10))

So, we make a fake tqdm class to make it work.

And it should be tracked by this issue: tqdm/tqdm#260 to avoid the conflict.

You SHOULD NOT IMPORT this class and keep it only working for qurry_progressbar() as type hint.
