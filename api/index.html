Qurrent#

Division for entangled entropy.

Qurrent/Qurshady - Second Renyi Entropy Measurement/Classical Shadow (qurry.qurrent)

Tip

  1. The name ‚Äúqurrent‚Äù was the second proposed name for this package. It‚Äôs not simply ‚Äúcurrent‚Äù with a ‚Äúq‚Äù replacing the ‚Äúc‚Äù, but rather a combination of ‚Äúqurr‚Äù and ‚Äúent‚Äù, where ‚Äúent‚Äù abbreviates ‚Äúentropy‚Äù, and ‚Äúqurr‚Äù is a coined prefix beginning with ‚Äúqu‚Äù to imply ‚Äúquantum‚Äù.

  2. The initial proposed name for the package was ‚Äúxproc‚Äù, short for ‚Äúexperimental process‚Äù.

  3. Another module, ‚Äúqurrech‚Äù, combines ‚Äúqurr‚Äù with ‚Äúech‚Äù, where ‚Äúech‚Äù is short for ‚Äúecho‚Äù, the Loschmidt echo being a key concept in quantum information theory. This inspired us to adopt ‚Äúqurry‚Äù, derived from ‚Äúqurr‚Äù and the suffix ‚Äúry‚Äù, to resemble words like ‚Äúquery‚Äù or ‚Äúcurry‚Äù.

  4. Since a package named ‚Äúqurry‚Äù already exists on PyPI, and many projects on GitHub also use this name, we changed the suffix from ‚Äúy‚Äù to ‚Äúium‚Äù, creating ‚Äúqurrium‚Äù. ‚ÄúQurrium‚Äù is unique and easily searchable, with no prior use on PyPI or in general web searches.

  5. So there is the evolution of the package name: xproc -> qurrent -> qurry -> qurrium

  6. Emoji: Qurry üçõ / Qurrium üìè
